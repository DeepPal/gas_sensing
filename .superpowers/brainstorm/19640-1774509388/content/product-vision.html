One critical question before designing the agentic AI layer:

The "agentic AI" can mean two very different things:

A — LLM-powered agents (Claude API)
The AI agent reasons in natural language, writes the methods section, explains anomalies, proposes next experiments in plain English. Uses Claude API internally.

B — Deterministic expert-system agents
Rule-based agents with ML models (GPR, CNN, Bayesian optimization). No LLM. Fast, offline, fully reproducible. The current approach, made more autonomous.

C — Hybrid (recommended)
Deterministic algorithms handle all signal math (fast, reproducible, no hallucination risk). Claude API handles high-level reasoning: anomaly explanation, experiment suggestions in English, methods text generation. Clearly separated layers.

Reply A, B, or C in terminal.
