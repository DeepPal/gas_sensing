Key structural decisions in this design

  • webapp/ replaces dashboard/ — same Python process, one port (8000), browser auto-opens on start
  • React frontend is built with Vite → static files served by FastAPI under / (no separate dev server in production)
  • All existing src/ code is kept — FastAPI routes call into the same SensorOrchestrator, RealTimePipeline, src/reporting/ modules
  • WebSocket endpoints are the existing live_server.py endpoints, moved into the unified FastAPI app
  • One command to start: lspr-tool start → uvicorn + opens browser + watchdog running
