# ADR Index

Architecture Decision Records (ADRs) capture major technical decisions, context, options considered, and consequences.

## Naming Convention

- File name format: ADR-XXXX-short-title.md
- Example: ADR-0002-realtime-pipeline-contract.md

## Required Sections

- Status
- Context
- Decision
- Alternatives Considered
- Consequences
- Migration/Implementation Notes
